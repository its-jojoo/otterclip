# otterclip
OtterClip is a frictionless clipboard vault for macOS/Windows/Linux: it captures everything you copy, deduplicates it, and lets you instantly search, pin, and paste via a global hotkey. Spotlight-like UX, fast fuzzy search, and privacy controls (pause + ignore patterns).

## Status
Early development (v0.1 target).

## v0.1 Goals
- Capture clipboard text (background)
- Deduplicate + max history limit
- Instant fuzzy search
- Global hotkey to open UI (Spotlight-like)
- Enter to paste, Esc to close
- Pins/favorites (never evicted)
- Privacy: pause capture + ignore patterns
