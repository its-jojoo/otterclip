<h1 align="center" style="margin-top:-50px;">OtterClip</h1>

<p align="center">
  A frictionless clipboard vault for macOS/Windows/Linux.
</p>

<p align="center">
  <i>Early development (v0.1).</i>
</p>

OtterClip captures everything you copy, deduplicates repeats, and lets you instantly search, pin, and paste via a global hotkey. Spotlight-like UX, fast fuzzy search, and privacy controls (pause + ignore patterns).

## v0.1 Goals
- Capture clipboard text (background)
- Deduplicate + configurable history limit
- Instant fuzzy search
- Global hotkey to open a Spotlight-like UI
- Enter to paste, Esc to close
- Pins/favorites (never evicted)
- Privacy: pause capture + ignore patterns
